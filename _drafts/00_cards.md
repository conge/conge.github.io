# 如何更重视对自己的承诺

大多数人，若与他人定下约会，都能按时赴约。可对若对自己订下承诺，却常常爽约。

你是否曾信誓旦旦说下誓言，要开始锻炼，冥想，写作，记日记，开展新项目……然后，却从无法坚守承诺……无疑，这会让你对自己失望。

在内心中，人们形成习惯，总是放自己一马。逐渐，就会发现，如果不涉及他人参与的约定，自己就无法做到说话算话。自己不比他人重要的信念，大约就是如此产生的吧。

这里，不是说你人品差，甚至不能说你有什么错。只是指出大多数人都有的倾向。注意到问题总是好的。

那，要怎么办呢？如何才能对自己也做到一诺千金？

我来列出一些重点吧：

首先，你得先有探究一下的好奇心。

先不要着急做改变，而是先要对当前情况保持好奇，研究一下现状。人们常常太过急于下判断，做改变，急于摆脱坏的方面，却不尝试先花时间了解一下自己。

先集中注意，观察或回忆，自己放自己鸽子的情况，都是如何发生了的。发生后，你的感觉怎样，又有怎样的想法？不要评判和逃避它们，转过身来，直面问题，你可否看清楚自己？

那些感觉和想法，你能否接受？能将它们作为你这个超凡的人的一部分？

你是否可以放下评判，与这些感觉和恐惧相处，而不需要它们消失？

这就是接受与爱，这就是我们探索其他可能性的起点。

## 重视自己的一些步骤

那，来看我们有什么其他方法呢？

这是我发现的一些有用的东西：

__与自己正式约会__. 是否有什么项目你想做，却总是推迟？与你自己预约个时间来完成它：把它放到你的日历上，认真对待这个约定，不要轻视它。不要让其他的事情打扰你，你已经定好了，这段时间属于你和你的这个项目。

__跟自己确认，你是否真的想做这事__。禅宗老师诺曼-费舍尔（Norman Fischer）说，承诺清晨冥想的过程从前一天晚上开始：问自己是否真的想做。如果你说是，那就再问一次。"确定吗？" 如果你说 "不"，那就把它从你的日程表上拿掉，然后去睡。但是，如果它对你很重要，你真的想做，那就真正承诺。

__把承诺当作神圣的事情__。前面说过，不要轻视。人们经常因轻视对自己的承诺，不顾后果的无所谓的将其推后。但请尝试把它当作神圣不可侵犯的约会；把它当作真正重要，超出平常的约定，把履约当作尊重自己和取悦自己的方式!

__带着好奇心、游戏、与欣赏的感觉__。不用紧张，你无须强迫自己做你不想做的事情。对自己的承诺，可以是一个借由好奇心驱使，让自己探索、游戏和学习的起点。它可以是一个愉悦的开端，是对自己和自己所从事的活动的欣赏。你愿意尝试，看看履行自己对自己的承诺会是怎样的吗？

尝试一下这些建议，尊重自己的重要性，告诉我你的体会吧。

* __工作过度__:过量工作，可能对尽可能多地完成工作，为了体会自己值得的感觉有所执念。
* __推迟拖延__：造成拖延的原因很多，其中之一可能是对想把事情做好，做对，打到特定效果的依恋。

# 堆积如山

当面对堆积的任务，我们常会，不知所措、灰心丧气。要么，就把它往哪儿一扔，决定碰都不碰。

这是再平常不过的反应了。人，都有过类似的时刻。

事务太多，解决掉它们太困难了，几乎是不可能完成的。杂事堆积，如同庞大的山，满是待付的账单，未回的电邮和短信，要勾划掉的待办事项清单。

一次解决一座山但想法，听上去好但实际上什么用也没有。我们啥时候也不可能挥挥手就解决一座山般但大量事情。我们只能一时一事，一步一步的来。

因此，别想着你需要搬山......你只想着下一件事，专注而快乐地面它。想象那些事情是你需要与之交谈的人。比如，你若虚与1000人谈话，你必须一个人一个人的倾听，不然，你就什么也听听不清......来，你能不能只专注于你面前的人，并真正与他共享当下？

享受登山的每一步，不要想着爬到山顶有多艰难。做好当下那一步，再一步，山顶自然会来到面前。

# 如何放开对过度思虑的沉迷

有时，我们的头脑会不停地想一些事情。那些想法盘旋脑际，纠缠不休，挥之不去。那些想法，或是关乎自己或他人，也可能是即将发生的大事，抑或是面对决定，无法当机立断。

这是人之常情。人难免面对关心之事担心沮丧，过度思虑，纠结不休。这个迹象表明，我们面对事件的不确定性而感到恐惧，而大脑正试图解决问题。

但，有时，放开这种过分思虑，能让人从中获得释放。停止过度思考，可以导致更好睡眠，心灵平静，以及生活简化。那么，我们如何才能做到放开过度思虑呢？

我来简要地分享个流程：

1. __觉察__。通常，我们陷入过度思虑的循环时，并不会意识到这一情形。持续思考和担心，似乎是唯一该做的事儿。但，若要走出这个循环，我们就必须觉察到我们正在做的事儿。有哪些迹象会指明你在一个循环中呢？对一些人来说，它是焦虑或担忧的程度增加，无法入睡，无法集中注意力。对其他人来说，可能是想对人发火，不停做计划，或想法设法获得控制权。关键是，你必须觉察到到你在过度思虑。

2. __面对背后的恐惧__。一旦你注意到自己的过分思虑，解决它的窍门，就是透过你所思虑事儿......去看其背后的恐惧。这就像揭秘魔术，不要看魔术师试图引导你注意的那只手，而是看他隐藏硬币的另一只。不要看让你忧虑的对象或需要你解决的问题......而是看让你产生纠结和过度思考的恐惧。你的身体，感觉怎样？
3. __体会恐惧__。在放开过度思虑之前，我们只能与我们身体中的恐惧感共处。你无法用忽视，去安抚被吓坏的孩子。试试恐惧同在，并与它一起放松，它放松下来吗？你能体察到恐惧，将其作为普通的感觉，并与它共处吗？你能深呼吸、慢呼吸，、放慢你的心跳，和恐惧一起放松吗？你能对它产生好奇心吗？如果这样做有挑战性，你可以试着想象它与你的心连在一起。
4. __探寻需求__。问问那恐惧，当下，它需要什么：一些关爱、音乐、一杯好茶、写日记或散步？然后问，回顾过度思考的时候，并问它那时需要什么——有哪些是简单的你马上能做的，用简单的语言说出心里所想？相信你得到的答案，深呼吸，从回忆转入当下。

5. __直面当下__。当下，你的面前，可有你能专注的事物？对我来说，它可以是房间里的光线、大自然中的花草树木清风白云、或一个人。或者，可以是回复电子邮件或写一篇文章。简单的，眼前的事情或物品。我用心选择一物或一事，相信这发自内心的选择，它足够帮我摆脱过分忧虑。然后，我专注于此时此刻此事。

这篇文章的本意，并不是说要事事完美，或者永远免除过度思虑或内心纠结。我们总会经常陷入思虑循环之中。我接受这是我人性的一部分，并热爱创造它的恐惧。

只是，我要练习更好的与之相处，并能做到在需要时，放手。
